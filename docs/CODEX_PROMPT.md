# Prompt جاهز لـ Codex

افتح هذا المشروع وراجعه بالكامل. المطلوب تحويله إلى نسخة إنتاجية نهائية لنظام دلفري يحتوي على تطبيق مندوب وتطبيق مطعم وBackend.

المطلوب بالتحديد:

1. شغّل backend وافحص كل endpoints.
2. شغّل تطبيق driver_app وتأكد من بناء APK.
3. شغّل تطبيق restaurant_app وتأكد من بناء APK.
4. أصلح أي أخطاء build أو runtime.
5. استبدل قاعدة JSON بـ PostgreSQL مع migrations.
6. أضف Docker Compose للـAPI وPostgreSQL.
7. أضف Firebase Cloud Messaging للتنبيهات.
8. حسّن تتبع الخلفية في تطبيق المندوب باستخدام Foreground Service.
9. أضف اختبارات API.
10. جهز build release ووقع APK إذا تم توفير keystore.
