# Production Checklist

قبل استخدام النظام تجاريًا:

- [ ] نقل قاعدة البيانات من JSON إلى PostgreSQL.
- [ ] تفعيل HTTPS.
- [ ] تغيير JWT_SECRET.
- [ ] إضافة نسخ احتياطي تلقائي.
- [ ] إضافة Firebase Cloud Messaging.
- [ ] تجربة التتبع على هواتف Android حقيقية.
- [ ] فحص استهلاك البطارية.
- [ ] إضافة سياسة خصوصية داخل التطبيق وعلى رابط خارجي.
- [ ] اختبار العقوبات يدويًا وتلقائيًا.
- [ ] اختبار محاسبة يوم كامل مع أكثر من مندوب.
- [ ] إضافة صلاحيات أدق للمدير والموظف والمحاسب.
- [ ] فحص أمني للكود والـAPI.
